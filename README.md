# 📝 Task Reminder Desktop Application

A Python-based desktop application that helps users manage and get reminders for their daily tasks. Built with Tkinter for the user interface, SQLite for local data storage, and Plyer for notification alerts.

## 📌 Features

- User registration and login system
- Add, update, and delete personal tasks
- Automatic desktop reminders for scheduled tasks
- Simple and clean GUI interface
- Lightweight and easy to run on any system

## 🛠 Technologies Used

- **Python**
- `Tkinter` – For GUI development  
- `Plyer` – For desktop notifications  
- `SQLite3` – For user and task data storage

## 📦 Setup Instructions

1. Clone the repository:
    ```bash
    git clone https://github.com/Bhavdiprathwa/Task-Reminder-Desktop-Application.git
    cd Task-Reminder-Desktop-Application
    ```

2. Install required dependencies:
    ```bash
    pip install plyer
    ```

3. Run the application:
    ```bash
    python app.py
    ```
